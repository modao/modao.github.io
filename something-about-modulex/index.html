modulex相关学习

Mar 29, 2015

什么是modulex

  • modulex 是新一代的模块加载器，可单独使用。其项目代码在github上
  • KISSY5将kissy拆分粒度足够细，使得所有的模块都可以方便的插拔。只要是modulex模块，就可以通过bower build到KISSY中
  • KISSY5允许自己构建适合自己的kissy
  • github上的kissy5项目，由于xtemplate的tag被删，使得bower install找不到xtemplate包

如何从KISSY1.4+升级到KISSY5

KISSY5对KISSY的模块进行了更进一步的拆分，同时还重命名了一部分模块（估计是作者对模块名称又思量再三），下面分享一下升级要修改的点：
  1. event模块拆分为引入event-dom模块，自定义事件改为引入event-custom模块
  2. uri模块改为引入url模块，并且里面的方法完全重写
  3. 原来挂载在KISSY上的mix、substitute等等方法要改为引入util模块之后再util.mix使用
  4. KISSY.use => require, KISSY.add => define, KISSY.config => require.config
  5. KISSY.param => querystring模块，QueryString.parse
